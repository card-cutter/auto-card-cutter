# 自動切卡工具 (Auto Card Cutter)

一鍵自動切卡工具 - iPhone專用

## 功能特性
- 🎯 自動檢測和裁邊 - 移除圖片周圍的白色/透明區域
- 🎴 自動分割 - 將圖片分割成卡牌（5列排列）
- 📦 批量處理 - 支持同時上傳多張圖片
- ⬇️ 一鍵下載 - 將所有卡牌打包成ZIP文件

## 使用方法
1. 點擊「選擇文件」按鈕上傳含有多張卡牌的圖片
2. 點擊「開始處理」按鈕
3. 工具會自動下載包含所有卡牌的 `cards.zip` 文件

## 使用線上版本
訪問：[Auto Card Cutter](https://card-cutter.github.io/auto-card-cutter/)

## 技術細節
- 純前端處理 - 無需服務器，保護隱私
- 使用Canvas API進行圖片處理
- 使用JSZip進行文件打包

## 卡牌規格
- 寬度比例：圖片寬度 ÷ 5
- 高度比例：寬度 × 1.4
- 輸出格式：PNG
